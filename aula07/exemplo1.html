Exemplo 1 - Estudo de cores no CSS

As cores no CSS podem ser utilizadas de varias maneiras, como por exemplo:

  • As cores podem ser representadas pelo nome, exemplo: red, green, blue, ...
  • As cores podem ser representadas atraves de uma função RGB, que é o vermelho,verde, e azul
  • AS cores, também pode ser representadas atraves de um número em hexadecimal
